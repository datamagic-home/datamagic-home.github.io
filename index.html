DataMagic

From Raw Data to Narrative Data Videos

VLDB 2026 Demo Submission

Abstract

Data videos integrate dynamic charts, voiceover narration, and synchronized animations to convey data insights through temporal storytelling. However, creating high-quality data videos requires expertise across data analysis, narrative design, and video production. Existing approaches either stop at static charts, require pre-prepared visual materials, or rely on pixel-level generation that cannot guarantee data fidelity.

We demonstrate DataMagic, an end-to-end interactive system that transforms raw tabular data and natural language queries into narrative data videos. The system introduces DVSpec (Data Video Specification), a declarative format that binds visual and animation elements to underlying data fields through semantic references, ensuring traceability. It employs a Generate-then-Orchestrate multi-agent architecture that generates candidate scenes in parallel before optimizing global narrative coherence.

System Architecture

DataMagic employs a two-stage Generate-then-Orchestrate architecture:

Input (CSV + Query) → Stage 1: Candidate Generation → DVSpec → Stage 2: Global Orchestration → Output Video

Stage 1: Candidate Generation

  • Story Planner: Query decomposition
  • Data Manager: Parallel data slicing
  • Visual Designer: Chart design & insights

Stage 2: Global Orchestration

  • Narration Director: Scene selection & ordering
  • Animation Coordinator: Audio-visual sync
  • Rendering Engine: Final video compilation

DVSpec serves as the declarative intermediate representation, decoupling logical description from rendering implementation. It enables data-driven semantic references and narration-indexed animation triggers.

DVSpec Example

Sample scene specification showing semantic references and narration-indexed triggers:

{ "type": "chart", "content": { "chart_type": "bar", "data_binding": {"x": "company", "y": "revenue"} }, "narration": [ "Nvidia leads with 122% growth", "followed by AMD at 89%" ], "animation": [ { "type": "highlight", "target": {"company": "Nvidia"}, // Semantic reference "trigger": {"narration_index": 0} // Auto-sync to audio } ] }

Interactive Demo Scenarios

Scenario 1: Automated Generation

Upload a CSV file (e.g., tech company financial data) and input a natural language query like "Analyze Q4 revenue peaks, regional marketing efficiency, and top product profit contributors." Watch the multi-agent engine decompose the query, extract data slices, design visualizations, and orchestrate a coherent narrative in real-time.

Scenario 2: Multi-Modal Editing

Refine generated videos through three interaction modes:

  • Canvas Manipulation: Click charts to switch types, adjust mappings, or modify colors
  • Script Editing: Edit narration text—animations auto-sync to new timing
  • Natural Language: "@Scene 5: change to pie chart and highlight top product"

Scenario 3: Data Q&A

Ask questions about the video: "Which product has the highest growth rate?" DVSpec's structured provenance maps visual elements to data fields, enabling precise retrieval from the original dataset. Convert Q&A insights into new video scenes: "Add a scene for regional sales distribution."
